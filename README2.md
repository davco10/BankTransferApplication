# Console-Based Balance Transfer System

## How It Works/Features
1. Allows user to select checking or savings account to transfer from and to
2. Allows user to enter a transfer amount
3. Program checks if user input is an int and greater than 0
4. Program subtracts from account transferred from and adds to account transferred to
5. Program checks if balance is less than or equal to low balance mode, if less than or equal to, balance will print red, if greater than low balance mode text will print original color 
5. Balances are updated and displayed
6. Transfer count is displayed, user is given 5 free transfers
7. Input will allow the user to make another transfer, if not program will stop running

## Technologies Used
- Python 3
- ANSI escape codes for terminal text coloring

## How To Run
1. Clone the repository
2. Run 'bank-transfer-application.py' using Python 3
3. Follow the prompts in the Console Log/Terminal
